# Vasant Jevengekar - Portfolio Website

A stunning, award-winning portfolio website built with pure HTML, CSS, and JavaScript. Features advanced animations, glassmorphism effects, 3D visuals, and a futuristic design that will impress recruiters and visitors.

## 🚀 Features

### Visual Design
- **Dark Futuristic Theme**: Modern dark theme with neon glow accents
- **Glassmorphism Effects**: Beautiful frosted glass card designs
- **3D Visuals**: Interactive 3D hover effects and transformations
- **Animated Backgrounds**: Floating blobs and particles
- **Gradient Animations**: Dynamic color gradients throughout
- **Smooth Transitions**: Fluid animations and micro-interactions

### Interactive Elements
- **Custom Cursor**: Animated cursor with follower effect
- **Scroll Animations**: Reveal animations triggered by scroll
- **Parallax Effects**: Multi-layer parallax scrolling
- **Progress Bars**: Animated skill progress indicators
- **Contact Form**: Functional form with validation
- **Social Links**: Animated social media buttons

### Sections
- **Hero Section**: Full-screen introduction with typing animation
- **About Section**: Professional about with internship highlights
- **Skills Section**: Technical skills with animated progress bars
- **Services Section**: Service cards with hover effects
- **Contact Section**: Split-screen contact form

### Performance & UX
- **Fully Responsive**: Mobile, tablet, and desktop optimized
- **Smooth Scrolling**: Butter-smooth scroll navigation
- **Loading Animation**: Professional loading screen
- **Back to Top**: Floating scroll-to-top button
- **Scroll Progress**: Visual scroll progress indicator

## 🛠️ Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Advanced animations, flexbox, grid, glassmorphism
- **Vanilla JavaScript**: No frameworks, pure JS interactions
- **Font Awesome**: Icon library
- **Google Fonts**: Inter and Space Grotesk typography

## 🚀 Getting Started

1. **Clone or download** the project files
2. **Open `index.html`** in your web browser
3. **No build process required** - pure HTML/CSS/JS

### Local Development
For local development with live reload:

```bash
# Using Python
python -m http.server 8000

# Using Node.js (if you have http-server)
npx http-server

# Using PHP
php -S localhost:8000
```

Then visit `http://localhost:8000` in your browser.

## 🎯 Customization

### Colors
Edit CSS variables in `css/styles.css`:
```css
:root {
    --primary-color: #00d4ff;
    --secondary-color: #ff00ff;
    --accent-color: #00ff88;
    /* ... */
}
```

### Content
Update the content in `index.html`:
- Personal information in hero section
- About text and internships
- Skills and percentages
- Contact details

### Animations
Modify animation speeds and effects in `css/animations.css` and `js/animations.js`.
